# E-Commerce-Sales-Customer-Insights-Dashboard
## Overview
This project is a **React-based E-Commerce Dashboard** that visualizes sales and customer insights. It allows users to explore monthly sales trends, top-selling products, and customer segmentation. This project demonstrates skills in React, CSS, and data visualization, making it ideal for data analyst and frontend portfolio showcases.

---

## Features
- Interactive dashboard built with React components
- Visualizes monthly sales, product categories, and revenue trends
- Supports customer segmentation analysis
- Responsive design for desktop and mobile
- Easy integration with backend APIs or mock JSON data

---

## Tech Stack
- **Frontend:** React, JSX
- **Styling:** CSS / SCSS
- **Data:** JSON / Mock API
- **Version Control:** Git & GitHub
